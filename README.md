# AR Smart Glasses Memory Assistance System

## Project Overview

This project explores the development of an **Augmented Reality (AR) smart glasses system** designed to support individuals living with memory loss, dementia, or Alzheimer’s disease. The goal of the system is to assist users in everyday social interactions by providing real-time contextual information, reducing confusion, and supporting independence.

The system leverages **facial recognition, voice transcription, and AI-powered summarisation** to recognise people, recall previous conversations, and present relevant information directly through an AR display.

This repository currently contains project planning, documentation, and design concepts. **No implementation code has been written at this stage.**

---

## Problem Context

Individuals with memory-related illnesses often struggle to recognise familiar people, remember recent conversations, and maintain meaningful social connections. Existing solutions provide limited real-time support during daily interactions.

Advancements in **AR technology** and **artificial intelligence** create an opportunity to address these challenges in a more intuitive and personalised way.

---

## Solution

The solution is a pair of AR smart glasses that:

- **Recognise people using facial recognition**  
- **Transcribe conversations using voice analysis**  
- **Generate AI-based summaries of conversations**  
- **Store information securely for future recall**  
- **Display contextual prompts in real time through AR**  

A companion mobile application allows **family members or caregivers** to manage features and adjust settings to ensure user comfort.

---

## Key Features (Planned)

- **Facial recognition** for identifying known individuals  
- **Real-time conversation transcription**  
- **AI-generated conversation summaries**  
- **Secure cloud-based storage**  
- **On-screen prompts** showing name, relationship, and previous interactions  
- **Date and time display** for orientation  
- **To-do list and reminder functionality**  
- **Family and caregiver control** via external app  

---

## Target Users

- **Individuals** with early to mid-stage memory loss or dementia  
- **Family members** supporting affected individuals  
- **Caregivers and healthcare professionals**  
- **General users** seeking AR-based daily assistance
